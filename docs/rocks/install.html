Sitecore Rocksのインストール¶

Sitecore Rocks は Sitecore から提供される Visual Studio 用の拡張プラグインです。Sitecore Rocks を使うことで、開発用のSitecoreインスタンスとVisual Studioをシームレスに統合した開発を行うことができるようになります。

このページで、Visual Studio 2015 Community Edition に Sitecore Rocks をインストールする手順を記載します。

注釈

Sitecore Rocks は Visual Studioの拡張プラグインのため、プラグインをインストールできるVisual Studioのエディションを使用してください。

Visual Studio を起動します。ツールバーの 拡張機能と更新プログラム を選択します。

拡張機能と更新プログラムダイアログが表示されたら、左側のカテゴリーで、オンライン を選択し、右上の検索ボックスに、 Sitecore Rocks と入力してプラグインを検索します。

Sitecore Rocksが一覧に表示されたら、ダウンロードボタンをクリックします。

インストールの確認画面が表示されたら、インストール をクリックします。

インストールが完了すると、ダイアログの下側に 再起動を促すメッセージが表示されるので、 今すぐ再起動 ボタンをクリックして Visual Studio を再起動します。

再起動をすると、Install Template Wizard ダイアログが表示されるので、 OK をクリックします。

Sitecore Rocks のインストールに成功すると、Visual Studio のツールバーに Sitecore が表示されるようになります。

これでインストール完了です。

注釈

Sitecore Rocksを使用した開発の詳細は、トレーニングおよび、 ドキュメントサイト で紹介されています。
