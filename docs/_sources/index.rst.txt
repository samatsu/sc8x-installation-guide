Sitecore 8.x スタンドアロン環境セットアップガイド
================================================================
このドキュメントではSitecoreの開発者向けに Sitecore 8.x に対応したスタンドアロンインストレーション環境を開発用にセットアップする方法を記載します。

構築に利用した各ソフトウェアのバージョンは次の通りです。

  :OS: Windows Server 2012 R2 Standard
  :Detabase:    SQL Server 2014 Developer Edition
  :MongoDB:     3.2.17
  :Sitecore:    8.2-update5

.. note:: 開発対象のSitecoreバージョンと互換性のあるソフトウェアのバージョンは `Sitecore compatibility table <https://kb.sitecore.net/articles/087164>`__ を参照してください。
